1. Tổng quan ứng dụng

Solution gồm hai chương trình chạy cùng một thư mục:

File Vai trò
KeyManager.exe Chạy nền (icon ở khay hệ thống — system tray). Lắng nghe phím tắt, mở Settings, khởi động CaptureImage.
CaptureImage.exe Hiện khi bạn bấm phím tắt: chọn vùng màn hình → chụp → OCR & dịch bằng AI.
Lưu ý: Bạn chỉ cần tự chạy KeyManager.exe. CaptureImage được KeyManager tự động mở khi cần.

2. Yêu cầu hệ thống

  • Hệ điều hành: Windows 10 hoặc Windows 11 (64-bit khuyến nghị)
  • .NET Framework: 4.6.2 trở lên (thường đã có sẵn trên Windows 10/11)
  • Internet: Bắt buộc khi bấm Convert to Text (gọi Google Gemini API)
  • Google AI API Key: Bắt buộc — xem mục 5 bên dưới
  • Quyền: Không cần quyền Administrator; cần quyền ghi Registry cho user hiện tại

3. Các file cần có trong thư mục cài đặt

Sau khi build hoặc copy bộ cài, thư mục (ví dụ Shared\) cần có:

File Bắt buộc Mô tả
KeyManager.exe Có App chính — chạy nền
CaptureImage.exe Có App chụp màn hình & OCR
CaptureImage.exe.config Có File cấu hình runtime của CaptureImage
Prompt.txt Có Prompt gửi cho AI (OCR + dịch + định dạng HTML)
helper.html Khuyến nghị File hướng dẫn này (mở từ menu Help)
Các file *.dll Có Thư viện phụ thuộc (NuGet) — copy cùng khi build
KeyManager.exe và CaptureImage.exe phải nằm cùng một thư mục. Nếu thiếu CaptureImage.exe, phím tắt sẽ báo lỗi.

4. Chạy lần đầu — từng bước

1 Mở thư mục chứa KeyManager.exe và double-click để chạy.

2 Kiểm tra system tray (góc dưới phải màn hình). Nếu không thấy icon, bấm mũi tên ^ (Show hidden icons) để tìm icon KeyManager.

3 Click phải vào icon tray → chọn Settings.

4 Nhập Google API key, chọn AI model (mặc định models/gemini-2.5-flash là ổn), kiểm tra Shortcut key (mặc định Alt+X).

5 Bấm Save.

6 Thử phím tắt (mặc định Alt+X): màn hình mờ → kéo chuột chọn vùng → thả chuột để chụp.

7 Trong cửa sổ kết quả, bấm Convert to Text và đợi AI xử lý.

Nếu thấy kết quả HTML với các section văn bản và nút Copy — cấu hình đã thành công.

5. Lấy Google API Key

  1. Truy cập Google AI Studio — API Keys (cần đăng nhập Google).
  2. Bấm Create API key (hoặc dùng key có sẵn).
  3. Copy chuỗi API key.
  4. Dán vào ô Google API key trong Settings → Save.
API key là thông tin nhạy cảm — không chia sẻ công khai. Ứng dụng lưu key trong Registry của user Windows (xem mục 8).
Nếu chưa nhập API key, CaptureImage vẫn mở được nhưng Convert to Text sẽ báo lỗi yêu cầu cấu hình Settings.

6. Cấu hình trong Settings

Mở Settings: Click phải icon tray → Settings

6.1. Shortcut key (phím tắt)

  • Hiển thị phím tắt hiện tại (mặc định: Alt+X).
  • Bấm Change... → nhấn tổ hợp phím mong muốn (phải có ít nhất một phím modifier: Ctrl, Alt, Shift hoặc Win).
  • Bấm Esc để huỷ khi đang ghi phím.
  • Sau khi Save, phím tắt có hiệu lực ngay — không cần khởi động lại KeyManager.
Nếu phím tắt trùng với ứng dụng khác, KeyManager hiện cảnh báo balloon "Hotkey Error". Hãy chọn tổ hợp phím khác.

6.2. AI model

  • Chọn từ danh sách hoặc gõ tên model (ví dụ: models/gemini-2.5-flash).
  • Nếu chỉ gõ tên ngắn (ví dụ gemini-2.0-flash), ứng dụng tự thêm prefix models/.
  • Model phải hỗ trợ đầu vào ảnh (multimodal) trên Google AI.

Các model gợi ý:

  • models/gemini-2.5-flash — mặc định, nhanh
  • models/gemini-2.0-flash
  • models/gemini-1.5-flash
  • models/gemini-1.5-pro — chất lượng cao hơn, chậm hơn

6.3. Google API key

  • Nhập key lấy từ Google AI Studio.
  • Dùng nút Show / Hide để xem hoặc ẩn key khi nhập.
  • Bắt buộc phải có trước khi OCR hoạt động.

6.4. Lưu cấu hình

Bấm Save để ghi vào Registry. Bấm Cancel để huỷ thay đổi.

Khởi động cùng Windows

KeyManager tự đăng ký chạy khi đăng nhập Windows (Startup). Sau khi reboot, icon tray sẽ xuất hiện lại — nhớ cấu hình Settings một lần là đủ.

8. Vị trí lưu cấu hình (Registry)

Cấu hình người dùng được lưu tại:

HKEY_CURRENT_USER\Software\AIOCRTRANS

Mở Registry Editor (regedit) và điều hướng tới path trên (không cần quyền admin).

Tên value Ý nghĩa
HotkeyModifiers Phím modifier (Alt=1, Ctrl=2, Shift=4, Win=8)
HotkeyKey Mã phím (mặc định 88 = phím X)
GoogleApiKey Google Gemini API key
AiModel Tên model AI (ví dụ models/gemini-2.5-flash)
Khuyến nghị chỉnh cấu hình qua Settings thay vì sửa Registry thủ công.

9. Xử lý sự cố

Phím tắt không hoạt động

  • Kiểm tra KeyManager đang chạy (icon trong tray).
  • Mở Settings → thử đổi phím tắt khác → Save.
  • Phím có thể bị ứng dụng khác chiếm (game overlay, Teams, v.v.).

Không thấy icon tray

  • Bấm mũi tên ^ trên taskbar để xem icon ẩn.
  • Kéo icon KeyManager ra vùng hiển thị thường xuyên (tuỳ cài đặt Windows).

Báo lỗi "CaptureImage.exe not found"

  • Đảm bảo CaptureImage.exe cùng thư mục với KeyManager.exe.
  • Build lại solution hoặc copy đủ file từ thư mục Shared\.

Convert to Text báo chưa có API key

  • Tray → Settings → nhập Google API key → Save.
  • Đóng CaptureImage (nếu đang mở) và bấm phím tắt lại để mở instance mới.

AI trả về lỗi hoặc không có text

  • Kiểm tra kết nối Internet.
  • Kiểm tra API key còn hiệu lực và có quota trên Google AI Studio.
  • Thử đổi model trong Settings (ví dụ sang gemini-2.0-flash).
  • Ảnh chụp quá mờ hoặc không có chữ — thử chụp vùng rõ hơn.

Convert to Text lỗi lần thứ 2 trở đi

  • Cập nhật lên bản CaptureImage mới nhất (đã sửa lỗi dispose ảnh).

Help không mở được

  • Đảm bảo file helper.html nằm cùng thư mục với KeyManager.exe.
  • Mở thủ công bằng trình duyệt: double-click helper.html.
